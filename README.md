# bokeh_experiments
Примеры кода для практического использования пакета Bokeh
* Реализовано два способа запуска приложений использующих Bokeh через сервер Flsk.  
  * Первый способ классический. Создается приложение app, <br>
      папка с приложением со всеми служебными папками.<br>
  * Второй способ через tornado. Пример показан в файле flask_embed.py<br>
